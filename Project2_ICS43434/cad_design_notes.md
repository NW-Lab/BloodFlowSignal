# ICS-43434マイク用ケース設計メモ

## ブレイクアウトボードの寸法

Adafruit ICS-43434 ブレイクアウトボード（製品番号: 6049）の寸法：
- 基板サイズ: 約16.5mm × 12.7mm
- 厚さ: 約1.8mm
- 取り付け穴: 直径約2.5mm（4隅）
- マイク位置: 基板中央付近
- 音孔（ポート）: 基板底面（ボトムポート）

参考: https://learn.adafruit.com/adafruit-i2s-mems-microphone-breakout/downloads

## 設計コンセプト

### 目的
血流音を計測するため、マイクを皮膚に直接当てて使用するケース

### 設計方針
1. **直接接触型**: マイクの音孔が皮膚に直接当たる構造
2. **遮音性**: 周囲の環境音を遮断
3. **安定保持**: 手で持ちやすく、測定中に動かない
4. **適度な圧力**: 皮膚に軽く押し当てられる

## ケース設計仕様

### 全体構造
- 形状: 円筒形ベース + 持ち手
- 材質想定: PLA/ABS（3Dプリント）

### 寸法
- ベース部直径: 30mm（皮膚接触面）
- ベース部高さ: 10mm
- 持ち手部直径: 20mm
- 持ち手部高さ: 40mm
- 全体高さ: 約50mm

### マイク固定部
- 基板収納スペース: 17mm × 13.5mm × 2mm（余裕を持たせる）
- 音孔開口部: 直径5mm（マイク位置に合わせる）
- 固定方法: スナップフィット or ネジ固定

### 接触面
- 平面形状（皮膚に均等に当たる）
- 音孔開口部のみ貫通
- エッジは面取り（皮膚への刺激を軽減）

### ケーブル引き出し
- 側面に切り欠き（ケーブル用）
- 幅: 5mm

## 設計パラメータ（調整可能）

```python
# 基本寸法
base_diameter = 30.0  # ベース部直径 (mm)
base_height = 10.0    # ベース部高さ (mm)
handle_diameter = 20.0  # 持ち手直径 (mm)
handle_height = 40.0   # 持ち手高さ (mm)

# マイク基板
pcb_width = 17.0      # 基板幅 (mm)
pcb_length = 13.5     # 基板長さ (mm)
pcb_thickness = 2.0   # 基板厚さ (mm)

# 音孔
port_diameter = 5.0   # 音孔直径 (mm)

# 壁厚
wall_thickness = 2.0  # 壁の厚さ (mm)
```

## 製造方法
- 3Dプリント（FDM方式）
- 推奨積層ピッチ: 0.2mm
- サポート材: 必要に応じて使用
